A Resta de duas funções lineares

A subtração de duas funções lineares resulta noutra função linear. Se tiver duas funções lineares, como f(x) = ax + b e g(x) = cx + d, a sua subtração h(x) = f(x) - g(x) resulta numa nova função h(x) = (a-c)x + (b-d), que mantém a forma linear mx + n onde m = a-c e n = b-d. Passos para subtrair funções lineares Identifique as funções: Comece com duas funções lineares, por exemplo, f(x) = ax + b e g(x) = cx + d. Subtraia os termos: Subtraia g(x) de f(x): f(x) - g(x). Reorganize os termos: Agrupe os termos semelhantes (termos com x e termos constantes): (ax - cx) + (b - d). Simplifique: Fatore e simplifique para obter a nova função: (a-c)x + (b-d).
